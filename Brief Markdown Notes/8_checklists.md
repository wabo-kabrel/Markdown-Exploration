#  Checklists
## Basic Task Lists
- [x] Completed task
- [ ] Incomplete task
- [x] Another completed task
- [ ] Task in progress
  
## Nested Checklists
- [x] Phase 1: Planning
  - [x] Requirements gathering
  - [x] Architecture design
  - [ ] Timeline creation
- [ ] Phase 2: Development
  - [ ] Backend development
  - [ ] Frontend development
  - [ ] Testing

## Real-World Example

### Project Checklist

- [x] Initialize repository
